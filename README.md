## üöÄ Installation

The recommended way to install Jacktook and its components is via this repository to ensure you receive automatic updates.

### 1. Add Source to Kodi
1. Open Kodi and go to **Settings** (gear icon).
2. Go to **File Manager** -> **Add source**.
3. Click on `<None>` and enter the following URL:
   `https://sam-max.github.io/repository.jacktook`
4. Name the source `Jacktook` and click **OK**.

### 2. Install Repository
1. Go back to the Kodi Home Screen.
2. Select **Add-ons** -> **Add-on browser** (open box icon).
3. Select **Install from zip file**.
4. Choose the `Jacktook` source you just added.
5. Select `repository.jacktook-x.x.x.zip`.

### 3. Install Jacktook
1. Select **Install from repository**.
2. Select **Jacktook Repository**.
3. Go to **Video add-ons** -> **Jacktook** -> **Install**.
